Box helps business of all sizes in every industry securely collaborate and manage their content in the cloud. This integration will import Box files into Slack.
•To import files from your Box account, use the Add File button in Slack (its beside the message input), or just include a shared link to the file in a message.
•Imported files will be automatically updated when the corresponding Box file is updated.
•Imported files are searchable and shareable but will continue to be stored in your Box account.
